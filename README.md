# Indkøb 🛒

En minimalistisk, synkroniseret indkøbsliste til hele familien.

**Live app:** https://gehrtmona-prog.github.io/indkob/

## Funktioner
- 🔄 Realtidssynk mellem alle, der deler en liste (via Supabase)
- 📱 Installerbar som app (PWA) — "Føj til hjemmeskærm"
- 👨‍👩‍👧 Inviter familien med ét link
- 📶 Virker offline; ændringer synkes, når der igen er net
- 📋 Flere lister med navne
- 🌗 Lyst/mørkt tema følger systemet

## Teknik
- Ren HTML/CSS/JS i én fil (`index.html`), ingen byggeproces
- Backend: Supabase (tabellen `items`)
- Hosting: GitHub Pages
